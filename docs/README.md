# Документация RMRP

Структура каталога `docs/` после реструктуризации:
- `overview/` — обзорные материалы (архитектура, функционал, презентация).
- `operations/` — эксплуатация, деплой и администрирование инфраструктуры.
- `planning/` — стратегические планы развития, разбитые по этапам.
- `specs/` — продуктовые спецификации и детальные требования.
- `archive/` — устаревшие материалы, сохранённые для истории.

Каждый документ содержит дату последнего обновления и владельца (при наличии), что упрощает ревизию актуальности.
